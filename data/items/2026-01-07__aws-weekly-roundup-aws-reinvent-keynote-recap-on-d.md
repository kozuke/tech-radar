# AWS Weekly Roundup: AWS re:Invent keynote recap, on-demand videos, and more (December 8, 2025)

## 要点
- AWS re:Invent 2025のキーノートでは、AIエージェントの進化と開発者の役割が強調された。
- Matt Garmanは開発者をAWSの「心臓」と位置づけ、AIエージェントがビジネスに具体的な利益をもたらすと述べた。Werner Vogelsは「ルネサンス開発者」の概念を提唱し、AI時代における開発者の進化を促した。

## 技術的ポイント
- **AIエージェント**: 自然言語で記述されたタスクを実行し、コードを生成、必要なツールを呼び出し、完全なソリューションを実行する。
- **Kiro Autonomous Agent**: 複数のリポジトリにまたがるバグのトリアージやコードカバレッジの改善を自動化するAI開発環境。
- **Multimodal Retrieval for Bedrock Knowledge Bases**: テキスト、画像、音声、ビデオファイルなど、様々なメディアタイプを対象としたAI検索・質問応答アプリケーションを構築可能。
- **AWS Interconnect – Multicloud**: Amazon VPCと他のクラウド環境（まずはGoogle Cloud、後にMicrosoft Azure）との間に、セキュアで高速なネットワーク接続を確立。

## 影響・アクション
- AIエージェントを活用した開発効率の向上が期待できる。
- Bedrock Knowledge BasesのMultimodal Retrievalを調査し、自社の検索・ナレッジマネジメントシステムへの応用を検討する。
- AWS Interconnect – Multicloudのプレビューを試し、マルチクラウド環境でのネットワーク接続を検証する。
