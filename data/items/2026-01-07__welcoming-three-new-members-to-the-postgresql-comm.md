# PostgreSQL Community Code of Conduct Committee に3名の新メンバーが参加

## 要点
- PostgreSQL Community Code of Conduct Committee に3名の新メンバー（Semab Tariq, Karen Riggs, Gary Evans）が加わった。
- コミュニティの行動規範を維持し、安全でインクルーシブな環境を促進することが目的。

## 技術的ポイント
- Semab Tariq: PostgreSQL関連ツールのパッケージング、Windows/macOS/Linux向けインストーラの構築経験を持つ。
- Gary Evans: エンタープライズ規模のPostgreSQL導入を支援し、オープンソースコミュニティとの連携を重視。
- 委員会メンバーは多様なバックグラウンド（DevOps, HR, エンタープライズサポート）を持つ。

## 影響・アクション
- コミュニティメンバーは、行動規範に関する懸念があれば coc@postgresql.org に連絡できる。
- PostgreSQLコミュニティの健全性維持に貢献する委員会メンバーの活動を認識する。
- 関連技術やドキュメント: PostgreSQL Community Code of Conduct (postgresql.org/about/policies/coc_committee/)
