# AWS Weekly Roundup: Amazon ECS, Amazon CloudWatch, Amazon Cognito and more (December 15, 2025)

## 要点
- 2025年12月15日の週のAWSアップデートのまとめ。Amazon ECS, Amazon CloudWatch, Amazon Cognitoなどの新機能が発表された。
- 開発者の生産性向上、セキュリティ強化、コスト削減に貢献するアップデートが含まれている。

## 技術的ポイント
- **Amazon WorkSpaces Secure Browser**: Webコンテンツフィルタリング機能が追加され、カテゴリベースのフィルタリングやURLポリシーによるWebアクセス制御が可能になった。
- **Amazon Aurora DSQL**: データベースのプロビジョニング時間が短縮され、数秒でクラスタ作成が可能になった。AWSコンソールクエリエディタやAIを活用した開発が容易になった。
- **Amazon Aurora PostgreSQL**: Kiro powersとの連携により、AI支援コーディングが可能になった。Kiro IDEからワンクリックでインストール可能。
- **Amazon ECS**: Fargateタスクでカスタムコンテナ停止シグナルがサポートされ、SIGQUITやSIGINTなどのシグナルを利用したコンテナの正常なシャットダウンが可能になった。
- **Amazon CloudWatch SDK**: JSONおよびCBORプロトコルがデフォルトとなり、レイテンシの低減、ペイロードサイズの削減、クライアント側のCPUおよびメモリ使用量の削減が実現された。
- **Amazon Cognito**: AWS PrivateLinkによるプライベート接続がサポートされ、認証トラフィックをパブリックインターネット経由でルーティングする必要がなくなった。
- **AWS Application Migration Service**: IPv6がサポートされ、IPv4とIPv6の両方をサポートするデュアルスタックサービスエンドポイントを使用してアプリケーションを移行できるようになった。

## 影響・アクション
- **実務への影響**: 各アップデートは、アプリケーション開発の迅速化、セキュリティの向上、インフラコストの最適化に役立つ。
